# 概要
flutter練習用に作成した、githubのレポジトリ検索アプリです。

# 機能
- githubレポジトリ検索
- リポジトリのweb表示
- お気に入り追加

# 開発環境
- flutter : 1.17.3
- Dart : 2.8.4
- OS : Windows 10 / MacOS Catalina
- IDE : Android Studio / Visual Studio Code
